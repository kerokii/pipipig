# Cat Daily Record Website

## Introduction
This repository contains the project documentation for the Cat Daily Record website, an application designed to help cat owners record and manage their cats' daily activities.

## Features
- Daily activity logs
- Medication reminders
- Feeding schedule management
- Health tracking and history
- User-friendly interface

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kerokii/pipipig.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pipipig
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application:
```bash
npm start
```
